## About

UniHarmony is a Python library that implements and provides several harmonization methods via an unified API.

## Installation

Use `pip` to install from PyPI like so:

```console
pip install uniharmony
```

You can also install via `uv`, like so:

```console
uv add uniharmony
```

# Features
## Data loading and simulation.
- Simulate multisite classification problems.
- Download and load MAREoS dataset

## Harmonization via interpolation models.
 - InterClassInterpolator (ICI)
