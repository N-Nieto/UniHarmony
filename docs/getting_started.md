# Getting Started

## Installation

UniHarmony is written in Python and [published on PyPI](https://pypi.org/project/uniharmony/). It is recommended to install it in a *virtual environment*.

### Install with `pip`

```console
pip install uniharmony
```

### Install with `uv`

```console
uv add uniharmony
```
